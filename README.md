# kilo-editor
JavaScript port for [kilo](https://github.com/snaptoken/kilo-src)

<img src="/publish/images/kilo-editor-demo.gif?raw=true" width="640px">


# How to install
```bash
$ npm -g i kilo-editor
```

# How to test
```bash
$ npm t
```
